Save New Album

Description:

Saves new Album in the album collection

HTTP Request:

POST https://hiptop.herokuapp.com/saveNewAlbum

Query Parameters

Parameter Default Description
id 700 Album id :Number(700 < id < 800)
author Album author :[String]
title Album title :String
urlSrc Album source :String
likes Album Likes :[Number](User's id)
genre Album genre :String
imgUrl Album image source :String
comment Album comments :[Number](Comment's id)
songId Album songs :[Number](song's id)

https://hiptop.herokuapp.com/saveNewAlbum

likes: 401, likes: 402, likes: 403, genre: pop, imgUrl: http://www,
comment: 801, comment: 802, comment: 803, songId: 402, songId: 403 }
